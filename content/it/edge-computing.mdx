## 엣지 컴퓨팅이란?

엣지 컴퓨팅(Edge Computing)은 데이터를 중앙 클라우드 서버가 아닌 데이터가 **생성되는 현장(엣지)** 근처에서 처리하는 기술입니다. 스마트폰, 공장 센서, 자율주행차, CCTV 등이 모두 엣지 디바이스입니다.

## 엣지 컴퓨팅이 필요한 이유

### 폭발적인 IoT 데이터

2030년까지 전 세계 IoT 기기 수는 약 750억 개에 달할 전망입니다. 이 모든 데이터를 클라우드로 전송하면:

- **대역폭 포화**: 네트워크 인프라 과부하
- **지연시간 문제**: 자율주행차는 0.1초의 지연도 치명적
- **전송 비용 급증**: 클라우드 데이터 전송 비용 폭발

<Callout type="warning">
  자율주행차는 초당 약 4TB의 데이터를 생성합니다. 이를 모두 클라우드에 전송하면 현재 5G 속도로도 처리가 불가능합니다.
</Callout>

## 5G와 엣지 컴퓨팅의 시너지

5G의 초저지연(1ms)과 초고속(20Gbps) 특성이 엣지 컴퓨팅과 결합하면 새로운 서비스가 가능합니다:

1. **실시간 AR/VR**: 클라우드 렌더링을 엣지에서 처리
2. **스마트 팩토리**: 생산 라인 이상 감지를 밀리초 단위로 처리
3. **원격 의료**: 원격 수술 로봇 제어
4. **자율주행**: V2X(Vehicle-to-Everything) 통신
