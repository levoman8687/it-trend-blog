## AI 에이전트란?

AI 에이전트는 **목표를 받으면 스스로 계획을 수립하고, 도구를 사용하며, 환경과 상호작용하여 목표를 달성**하는 AI 시스템입니다. 단순히 질문에 답하는 것을 넘어 실제로 작업을 수행합니다.

<Callout type="info">
  **간단한 비유**: 기존 LLM은 "이메일 어떻게 쓰나요?"에 답변합니다. AI 에이전트는 "내일 회의 일정을 팀원들에게 이메일로 보내줘"라고 하면 캘린더 확인 → 이메일 작성 → 발송까지 스스로 처리합니다.
</Callout>

## 주요 에이전트 프레임워크

### ReAct (Reasoning + Acting)

가장 기본적인 에이전트 패턴입니다. 생각(Reasoning)과 행동(Acting)을 반복합니다.

```
목표: "삼성전자 현재 주가를 찾아서 알려줘"

[생각]: 주가 정보는 실시간 검색이 필요하다.
[행동]: 웹 검색 도구 호출 → "삼성전자 현재 주가"
[관찰]: 검색 결과 = "₩72,000 (2026-02-19 기준)"
[생각]: 검색 결과를 얻었다. 사용자에게 알려주자.
[최종 답변]: 삼성전자 현재 주가는 72,000원입니다.
```

### AutoGPT / BabyAGI

더 복잡한 목표를 여러 단계로 분해하여 처리하는 다단계 에이전트입니다.

### Claude Computer Use

Anthropic의 Claude 3.5가 지원하는 기능으로, **실제 컴퓨터 화면을 보고 마우스·키보드를 제어**할 수 있습니다.

<Callout type="warning">
  **주의사항**: AI 에이전트는 강력한 만큼 위험도 있습니다. 파일 삭제, 이메일 발송 같은 비가역적 작업에는 인간의 최종 확인이 필요합니다.
</Callout>

## AI 에이전트의 실제 활용 사례

1. **소프트웨어 개발**: Devin (Cognition AI) - 전체 개발 작업 자동화
2. **고객 서비스**: 24시간 티켓 자동 처리, 환불 처리
3. **리서치 자동화**: 웹 검색 → 요약 → 보고서 작성
4. **데이터 분석**: 데이터 수집 → 분석 → 시각화 → 인사이트 도출

<Callout type="tip">
  **시작하기 좋은 방법**: LangChain + OpenAI API로 간단한 웹 검색 에이전트를 만들어보세요. 약 50줄의 파이썬 코드로 구현 가능합니다.
</Callout>
