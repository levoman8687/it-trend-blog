## 확산 모델(Diffusion Model)이란?

확산 모델은 이미지에 점진적으로 **노이즈를 추가하고(Diffusion), 다시 제거하는(Denoising) 과정**을 학습하여 새로운 이미지를 생성하는 AI 기술입니다. 2022년 Stable Diffusion 공개 이후 이미지 생성 AI의 주류가 되었습니다.

<Callout type="info">
  **작동 원리**: 수백만 장의 이미지-텍스트 쌍을 학습 → 텍스트 프롬프트를 받으면 순수 노이즈에서 시작하여 점차 의미 있는 이미지로 변환
</Callout>

## 각 모델 특징

### DALL-E 3

ChatGPT와 통합된 OpenAI의 이미지 생성 모델입니다.

**특징**
- ChatGPT에서 자연어로 이미지 요청 가능
- 텍스트를 이미지 내에 정확하게 렌더링 (간판, 로고, 명함)
- 부적절한 콘텐츠 필터링 강력

**최적 활용 사례**
- 텍스트가 포함된 마케팅 이미지
- 빠른 콘셉트 스케치
- 어린이 교육 콘텐츠

### Midjourney v7

예술성과 심미성이 가장 뛰어난 상업용 서비스입니다.

**특징**
- 사진과 구별이 어려운 수준의 사실적 이미지
- 다양한 예술 스타일 완벽 재현
- 커뮤니티 기반의 프롬프트 공유 문화

```
/imagine prompt: a futuristic Seoul cityscape at sunset,
cyberpunk aesthetic, flying cars, neon lights reflecting
on Han River, ultra detailed, 8k resolution --ar 16:9 --v 7
```

### Stable Diffusion 3.5

완전 오픈소스로 로컬에서 실행 가능한 모델입니다.

**특징**
- 인터넷 연결 없이 로컬 PC에서 실행
- LoRA, ControlNet 등으로 무한 커스터마이징
- 기업 내부 사용 시 데이터 유출 우려 없음

<Callout type="tip">
  **SD 로컬 실행 권장 사양**: NVIDIA RTX 4060 이상 GPU, VRAM 8GB 이상, RAM 16GB 이상
</Callout>
